## ✨ 功能概述

本技能实现豆包 AI 图片生成的自动化工作流，适用于以下场景：

| 场景 | 描述 |
|------|------|
| 🎬 海报设计 | 社交媒体海报、活动宣传图 |
| 🎨 插画创作 | 博客配图、文章封面 |
| 📌 贴纸设计 | 表情贴纸、IP 形象贴纸 |
| 👤 头像生成 | 个人头像、卡通形象 |

### 触发方式

当用户说出以下指令时自动执行：

- "生成 XX 海报"、"做一张 XX 图"
- "帮我画 XX"、"创建一个 XX 图片"
- "用豆包生成"、"在豆包做"
- "小麦帮我生成几张 XX"

## 🔄 自动执行流程

当用户触发本技能时，自动执行以下步骤：

### Step 0: 检查登录状态

```javascript
// 使用 browser 工具检查豆包登录状态
browser(action: snapshot)
// 如需登录，提示用户扫码
```

> 💡 如需登录，会提示：「需要你登录豆包，请扫码or点击登录，完成后告诉我」

### Step 1: 打开豆包

```javascript
// 使用 browser 工具打开豆包首页
browser(
  action: "open",
  url: "https://www.doubao.com",
  profile: "openclaw"
)
```

### Step 2: 新建对话

- 点击左上角 **「新的对话」** 按钮

### Step 3: 进入图片生成模式

- 点击输入框下方的 **「图像生成」** 图标（通常在发送按钮附近）

### Step 4: 输入 Prompt

- 在输入框中输入图片描述
- 参考下文的 [Prompt 编写指南](#-prompt-编写指南)

### Step 5: 等待生成

- 等待图片生成完成（通常 **几秒到十几秒**）
- 可用 `browser(action: snapshot)` 查看进度

### Step 6: 选择并下载

- 点击选中满意的图片
- 点击右上角 **「下载原图」** 按钮保存

## 📝 Prompt 编写指南

### 基础结构

推荐使用以下结构组合 Prompt：

```
[主体描述] + [风格] + [配色] + [细节] + [技术参数]
```

### 常用风格关键词

| 风格 | 关键词 |
|------|--------|
| 卡通/动漫 | 卡通风格，动漫风格，二次元，卡通插画 |
| 潮流 | 潮流风格，街头风，酷炫，扁平化设计 |
| 可爱 | 可爱风格，萌趣，Q版，卡通形象 |
| 写实 | 写实风格，真实感，高清，精细细节 |
| 国风 | 中国风，国潮，传统元素，水墨 |
| 极简 | 极简风格，简约，现代设计，简洁 |

### 贴纸类 Prompt 模板

```plaintext
[主题]贴纸，XX风格，[主体描述]，
[动作/表情]，[背景描述]，
卡通贴纸效果，PNG透明背景，干净背景
```

### 增效关键词

在 Prompt 末尾添加以下词汇可提升效果：

```plaintext
高清画质，精细细节
数字艺术作品
柔和的光影
干净的背景
```

## ⚠️ 注意事项

1. **中文 Prompt 效果更好** 🇨🇳
   - 豆包对中文理解更准确，建议使用中文描述

2. **分步描述** 📝
   - 复杂画面先描述主体，再补充细节

3. **明确风格** 🎯
   - 风格关键词要具体（如"二次元漫画风"而非"好看"）

4. **尺寸选择** 📐
   - 生成后可选择不同尺寸（16:9、1:1、9:16 等）

## ❓ FAQ

### Q1: 提示需要登录怎么办？

**A**: 豆包账号未登录时，会提示你扫码登录。完成登录后告诉 AI 即可继续执行。

### Q2: 生成速度慢怎么办？

**A**: 
- 检查网络连接
- 豆包服务器繁忙时可能需要等待
- 可用 `browser(action: snapshot)` 查看生成进度

### Q3: 生成结果不满意怎么办？

**A**: 
- 收集具体反馈（如颜色、风格、细节等）
- 重新生成时调整 Prompt
- 豆包支持多张图片选择，可挑选最满意的

### Q4: 支持哪些图片格式？

**A**: 豆包支持 PNG、JPG 等常见格式，下载时通常为 PNG 格式。

### Q5: 可以生成中文文字吗？

**A**: 可以，但在 Prompt 中明确写出需要文字内容（如"大字'禁毒'"），生成效果会更准确。
