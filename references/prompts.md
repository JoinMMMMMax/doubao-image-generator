# 常用Prompt模板库

## 按类型分类

### 贴纸/Sticker
```
[主题]贴纸，卡通风格，[主体描述]，
[动作/表情]设计，PNG透明背景，贴纸工艺效果
```

示例：
- 禁毒贴纸：禁毒主题贴纸，潮流酷风格少女，手持禁止标志，坚定表情，红色警示元素，扁平化设计
- 猫猫贴纸：萌趣猫咪贴纸，胖乎乎的卡通猫咪，治愈系表情，手举小花，浅色系，PNG透明

### 头像/Avatar
```
[描述]头像，[风格]风格，[主体描述]，
[表情/特点]，[服装]，[配色]
```

### 海报/Poster
```
[主题]海报，[风格]风格，[核心元素]，
[背景]，[配色方案]，[用途]
```

### 插画/Illustration
```
[主题]插画，[风格]，[主体描述]，
[场景]，[光线]，[氛围]
```

## 按风格分类

### 二次元/动漫风
```
XX主题二次元插画，日系动漫风格，
[角色描述]，大眼睛，长发，[动作]，
柔和配色，精致细节，动漫CG风格
```

### 潮流/街头风
```
XX主题潮流海报，街头风格，
[主体]，[配色]，[潮流元素]，
扁平化设计，矢量感，视觉冲击力强
```

### 可爱/萌系
```
XX主题可爱插画，Q版风格，卡通形象，
[主体]萌趣，[表情]，[小配件]，
浅色系配色，治愈系，温暖
```

### 国风/传统
```
XX主题国风插画，中国传统风格，
[元素]，水墨质感，[配色]，
传统与现代结合，优雅
```
